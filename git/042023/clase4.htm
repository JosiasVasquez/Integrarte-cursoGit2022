Cuarta sesión curso de control de versiones con git y github:

Temas vistos en la sesión:

  • Repaso,
  • Que es github,
  • Como crear cuenta en github,
  • Exploración por la interfaz de github,
  • Exploración y edición de perfil en github
  • Contenido de la página code de un repositorio,
  • El archivo readme.md,
  • que son las liveraciones (releases),
  • Que son las insidencias o issues,
  • Como crear una insidencia (issue),
  • Como clonar un repositorio por https y SSH,
  • Como crear una clave SSH,
  • Como agregar la clave SSH a nuestra cuenta en github,
  • Como agregar la clave GPG a nuestra cuenta en github,

Comandos vistos en la sesión:
gpg --export -a "Nombre de Usuario" > public.key
Exportar la clave pública gpg.
ssh-keygen -t ed25519 -C "correo@ejemplo.com"
Crear una clave SSH
ssh git@github.com
Comprobar la conexión SSH a github.
git clone https://dominio.com/usuario/repositorio.git
Clona un repositorio por https.
git clone git@github.com:usuario/repositorio
Clona el repositorio usando SSH.
